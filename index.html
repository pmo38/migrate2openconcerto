Cette documentation concerne la migration d’une comptabilité à partir d’un fichier FEC (Fichier des Écritures Comptables) vers une comptabilité sous le logiciel OpenConcerto.


Cette documentation se présente sous les formes :

  • une carte heuristique (mind map).

  • un export html fait à la racine du référentiel.

  • des ressources : scripts, schémas, fichiers pdf, vidéos.



Cliquez pour ouvrir la page html



La carte heuristique est faite avec le logiciel VYM (View Your Mind).


L’outil VYM permet de dérouler ou enrouler une partie de la carte. Cela permet de choisir le focus.


L’export html englobe toute la carte. C’est un peu comme avoir tout un livre présenté dans une seule page.

Si l’ergonomie du format html pose problème, c’est un pis aller pour ceux qui ne veulent pas installer VYM et cloner le dépôt.
